🖥️ Hardware Setup — The Engine Room

The golden rule: your GPU is your engine, VRAM is your fuel. Never compromise on this.

Why Nvidia? CUDA. The entire AI ecosystem (PyTorch, ComfyUI, llama.cpp) is optimized for CUDA. AMD GPUs work, but with more friction and less community support.

⚙️ Software Stack & Models

1. Python + Conda — The Foundation

Mastering your Python environments means mastering your machine. Conda lets you isolate each project with its own dependencies, no conflicts. Essential as soon as you're running multiple tools.

2. Ollama — Local LLMs Without Friction

The smoothest tool for deploying and interacting with language models locally. One command to download and launch any model.

Why MoE models? Mixture-of-Experts architectures activate only a fraction of parameters per token — you get near-flagship quality at a fraction of the VRAM cost.
  • Qwen3.6-35B-A3B (MoE + MTP) — Outstanding reasoning, coding, and multilingual support. My main choice. Runs on 16 GB VRAM.
  • Gemma 4 26B-A4B (MoE) — Google's MoE flagship. Excellent instruction-following and low memory footprint.

3. ComfyUI — Images & Video With Full Control

The reference node-based interface for image and video generation. The learning curve is real, but the level of control is unmatched.

Images:

  • FLUX.2 Klein 9B — Current state of the art for open-source image generation. Best prompt adherence and photorealism available locally.
  • Z Image Turbo — Ultra-fast generation (seconds) without sacrificing quality. Ideal for rapid iteration.
  • SDXL + fine-tuned LoRAs — Unbeatable ecosystem depth. Thousands of styles and characters available on CivitAI.

Video:

  • Wan 2.2 — My go-to for image animation. Remarkable motion quality and temporal consistency.

4. llama.cpp — Going Further

When Ollama isn't enough, llama.cpp gives granular control over generation parameters, threading, and GPU/CPU usage. Essential for integration into automated workflows.

5. Stable Audio — Local Music Generation

Generate ambient music directly from a text prompt, fully locally. Stable Audio 3 produces high-quality stereo audio up to several minutes long — ideal for video backgrounds and reels.

🤖 Automation — Where It Gets Powerful

The real value of local AI is automation. Where cloud tools lock you into a web interface, local gives you full API and scripting access to everything.

What I've automated in my workflow:

  • Content generation — LLM → image/video prompt → ComfyUI generation
  • Post-production — TTS audio, background music, karaoke subtitles
  • Publishing — scheduling, multi-account management, platform-adapted formats
  • News monitoring — source scraping → LLM summary → formatted post

The whole pipeline runs without human intervention, 24/7.

Tool I use to build workflows: Claude (Anthropic) — for architecture design, code generation, and prompt refinement. I use it as a development assistant (cloud-based), even though all production runs locally.
🚀 Where to Start

The Fast Lane — Pinokio

Pinokio installs AI tools in one click. Perfect for testing ComfyUI or Ollama without touching the command line.

The Pro Route — Manual Installation

Clone repos directly from GitHub for full control over versions, parameters, and optimizations.

  1. Latest Nvidia drivers
  2. CUDA Toolkit (compatible with your target PyTorch version)
  3. Miniconda
  4. Ollama (quick test to validate GPU setup)
  5. ComfyUI
  6. llama.cpp (optional, for advanced workflows)
